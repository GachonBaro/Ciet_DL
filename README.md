# SECO_Deep Learning
This project was tested using the 'movielens 100k' dataset to create a recommendation algorithm to be applied to SECO.

## Requirements
- Dataset

   - URL: https://grouplens.org/datasets/movielens/100k/ (Be aware of copyright)
